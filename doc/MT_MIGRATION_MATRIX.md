# MT Migration Matrix

本文档用于对照原始仓库 `JuchiaLu/Multi-Supplier-MT-Plugin` 的 MT 能力与当前 `memoq-ai-tool` 桌面端实现状态，并作为后续迁移基线。

状态说明：

- `已迁移`：桌面端已成为权威实现。
- `部分迁移`：桌面端已有入口或字段，但尚未完整承接。
- `未迁移`：能力仍缺失。
- `明确放弃`：不适合当前桌面架构，且已确认不迁移。
